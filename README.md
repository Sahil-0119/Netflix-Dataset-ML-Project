# Netflix-Dataset-ML-Project
A machine learning project that analyzes the Netflix titles dataset to classify content as Movies or TV Shows using supervised learning techniques such as Logistic Regression, Decision Trees, KNN, and Naive Bayes. Includes data preprocessing, feature engineering, model evaluation, and visualizations.
